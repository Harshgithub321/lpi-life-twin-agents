# \# Life Optimization AI Agent

# 

# \## What it does

# This agent helps users improve their daily life using SMILE methodology.

# 

# \## Features

# \- Accepts user input

# \- Uses LPI tools (smile\_overview + get\_insights)

# \- Runs local LLM (Ollama)

# \- Provides personalized advice

# \- Explains data sources (Explainable AI)

# 

# \## How to run

# 1\. Start LPI server

# 2\. Run ollama serve

# 3\. python agent.py "your input"
